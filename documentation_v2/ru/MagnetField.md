# Подробное описание класса: MagnetField

## 1. Общая информация
*   **Имя класса:** `MagnetField`
*   **Тип:** `Activity` (Экран)
*   **Назначение:** Эта Activity измеряет окружающее магнитное поле вокруг устройства. Ее можно использовать для определения магнитных полюсов Земли (как компас) или для поиска находящихся поблизости магнитов и металлических предметов.
*   **Взаимодействие:** Запускается из `MainActivity`. Она автоматически начинает измерения при открытии и прекращает их, когда пользователь покидает экран.

## 2. Переменные (Поля класса)

| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `sensorManager`| `SensorManager` | Системный сервис для управления датчиками. | В `onCreate()` и `onStop()`. |
| `magneticFieldSensor`| `Sensor` | Ссылка на оборудование магнитометра. | В `onCreate()`. |
| `magneticFieldListener`| `SensorEventListener`| Обрабатывает входящие магнитные данные. | Определено в `onCreate()`. |
| `tvMX, tvMY, tvMZ`| `TextView` | Отображает напряженность магнитного поля по осям X, Y, Z. | В `onSensorChanged()`. |

## 3. Методы класса

### Имя метода: `onCreate`
*   **Что делает метод:**
    1. Устанавливает разметку.
    2. Инициализирует текстовые поля.
    3. Находит датчик `MAGNETIC_FIELD`.
    4. Определяет слушателя: считывает `values[0, 1, 2]` (измеряются в **микротеслах, мкТл**) и отображает их на экране.
    5. Сразу регистрирует слушателя, чтобы измерения начались немедленно.
*   **Когда вызывается:** При запуске Activity.

### Имя метода: `onStop`
*   **Что делает метод:** Отменяет регистрацию магнитного слушателя.
*   **Когда вызывается:** Когда Activity больше не видна пользователю.
*   **Важно:** Использование `onStop` вместо `onPause` допустимо, но `onPause` обычно предпочтительнее для более быстрого освобождения ресурсов.

## 4. Жизненный цикл
*   **`onCreate()`**: Инициализация и запуск датчика.
*   **`onStop()`**: Остановка датчика для экономии энергии.

## 5. Взаимодействие с интерфейсом (UI)
*   **Элементы:** 3 текстовых поля (`TextView`).
*   **События:** Нет (автоматическое обновление).

## 6. Общая логика
1. Пользователь открывает экран магнитометра.
2. Телефон фиксирует магнитное притяжение окружающей среды.
3. Значения X, Y и Z обновляются, когда пользователь подносит телефон к куску металла или поворачивает его к Северному полюсу.

## 7. Упрощенное объяснение
**«Объяснение простыми словами»**
Представьте, что в ваш телефон встроена **стрелка компаса**. Этот класс считывает, как эту стрелку тянут невидимые магнитные силы. Если вы поднесете кухонный магнит близко к телефону, вы увидите, как цифры резко подскочат!

---
**Заметка для студентов:** Напряженность магнитного поля измеряется в микротеслах. Нормальный фон Земли обычно составляет от 25 до 65 мкТл.
